Usage

Shorten a URL:

$ curl -X POST http://share.andrewgarner.com \
--user user:pass -i \
-F"url=https://github.com/andrewgarner/share.andrewgarner.com"
...
Location: http://share.andrewgarner.com/o2pbXA

Shorten a URL with custom code:

$ curl -X POST http://share.andrewgarner.com \
--user user:pass -i \
-F"url=https://github.com/andrewgarner/share.andrewgarner.com" \
-F"code=gh"
..
Location: http://share.andrewgarner.com/gh
